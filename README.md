MySite_MEAN
===========
My personal site with MEAN stack

Dependencies
-----------
### Backend
   - NodeJS
      - connect-multiparty
   - Express framework
      - morgan
      - body-parser
      - method-override
      - cookie-parser
   - MongoDB
      - mongoose

### Frontend
   - AngularJS
     - angular-ui-router
     - angular-resource
     - angular-file-upload
   - Bootstrap
   - FontAwesome
