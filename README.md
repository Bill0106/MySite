MySite_MEAN
===========
My personal site with MEAN stack

Dependencies
------------
### Backend
- NodeJS
  - connect-multiparty
  - request
  - cheerio
  - moment
- Express framework
  - morgan
  - body-parser
  - method-override
  - cookie-parser
- MongoDB
  - mongoose

### Frontend
- AngularJS
  - angular-ui-router
  - angular-resource
  - angular-file-upload
  - ngInfiniteScroll
- Bootstrap
- FontAwesome
