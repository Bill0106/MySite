MySite_MEAN
===========
My personal site with MEAN stack

Dependencies
-----------
### Backend
   - NodeJS
      - connect-multiparty
   - Express framework
      - morgan
      - body-parser
      - method-override
      - cookie-parser
   - MongoDB
      - mongoose

### Frontend
   - AngularJS
     - angular-route
     - angular-animate
     - ngInfiniteSroll
   - Bootstrap
   - FontAwesome
   - animate.css
   - Html5shiv
   - Respond.js
